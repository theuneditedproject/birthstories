Sign in to admin

Enter your GitHub repo info and a Personal Access Token. Both are stored in this browser only — they never leave your device except to talk to GitHub.

How do I get a Personal Access Token?
  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Click "Generate new token"
  3. Token name: unedited-admin
  4. Expiration: pick something reasonable (1 year is fine)
  5. Repository access: "Only select repositories" → choose your unedited repo
  6. Repository permissions: set Contents to Read and write
  7. Click "Generate token" and copy the token (starts with github_pat_)
  8. Paste it above and sign in

The token is stored only in your browser's localStorage. To sign out and remove it, click "Sign out" once logged in.
